# Assets Directory

Organized folder structure for all website assets:

## Folder Structure

### `/logos`
- `makamithi.png` - Main company logo
- `makamithi-white.png` - White version for dark backgrounds
- `favicon.ico` - Website favicon

### `/images`
- General website images
- Hero section backgrounds
- About section images

### `/products`
- Animal feeds images
- Fertilizers images
- Seeds images
- Crop protection images
- Veterinary products images

### `/partners`
- Partner company logos (e.g., Bayer, Unga, Syngenta, Corteva, Seedco, Osha)

### `/css`
- External stylesheet files (if needed)

### `/js`
- External JavaScript files (if needed)

## Current Setup

The website currently uses:
- Font Awesome icons for all visual elements
- Unsplash CDN for product and background images
- Google Fonts for typography (Inter, Arizonia)

## Usage

Place your images in the appropriate folders:
- Logo: `assets/logos/makamithi.png`
- Products: `assets/products/product-name.jpg`
- Partners: `assets/partners/partner-name.png`
